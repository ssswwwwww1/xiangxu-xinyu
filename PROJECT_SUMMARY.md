# 乡叙心域 - AI 乡村疗愈平台项目总结

## 📋 项目概述

**乡叙心域**是一个创新的 AI 驱动乡村疗愈平台，通过 AI 技术弥合传统村落的文化资源与都市人群心理需求之间的鸿沟，让沉默的文化"开口说话"，治愈现代人的"心病"。

项目核心理念：**从"观光"到"疗愈"的智慧桥梁**

---

## ✨ 核心功能实现

### 1. AI 辅助智能叙事生成引擎
- ✅ 基于用户情绪测评结果生成个性化路线
- ✅ 支持 3 种不同情绪类型的路线（宁静/活力/文化）
- ✅ 每个路线包含 3 个精心设计的打卡点
- ✅ 结合天气、时间等实时数据动态调整

### 2. AI 情感计算与分析引擎
- ✅ 5 道心理测评题精准评估用户情绪状态
- ✅ 实时分析用户在打卡中的情感表达
- ✅ 支持 6 种情感标签（平静/喜悦/感动/启发/怀念/专注）
- ✅ 情感可视化统计和分布展示

### 3. AI 生成式内容引擎
- ✅ 自动生成"心路影像诗"
- ✅ 整合用户所有碎片化记录（照片/语音/文字）
- ✅ 支持分享和下载为线装手账
- ✅ 旅程统计数据可视化

### 4. AI 驱动的情感化数字人
- ✅ 定制化数字人向导"福宝"
- ✅ 智能对话和情感回应
- ✅ 语音和文字双模交互
- ✅ 表情和动作的情感化表达

## 🎨 页面设计

### 1. 首页 (HomePage)
- 精美的渐变背景和动画效果
- 四大核心引擎功能展示
- 引导用户开始疗愈之旅的 CTA

### 2. 情感测评页 (AssessmentPage)
- 渐进式问题设计，避免用户疲劳
- 精美的情感表情符号选择
- 实时进度条和 AI 分析动画

### 3. 叙事路线页 (JourneyPage)
- 个性化路线推荐展示
- 详细的打卡点信息和活动说明
- 实时天气和时间显示
- 情感标签和预计时长

### 4. 打卡互动页 (CheckInPage)
- 三标签设计：拍照/语音/文字
- 情感标签选择器
- 实时预览和提交反馈

### 5. 心灵游记页 (MemoryPage)
- AI 生成诗歌展示
- 旅程统计和情感分布可视化
- 打卡记录时间线
- 分享和下载功能

### 6. 数字人页 (DigitalHumanPage)
- 情感化数字人形象
- 智能对话界面
- 快捷问题推荐
- 语音交互支持

## 🚀 启动方式

### 方式一：完整开发环境

1. **安装依赖**
```bash
npm install
```

2. **启动前端**
```bash
npm run dev
```

3. **启动后端（可选）**
```bash
npm run server
```

4. **访问** http://localhost:5173

直接打开 `demo.html` 文件即可在浏览器中运行，无需任何安装！

## 🎯 核心矛盾解决

### 矛盾点
传统村落拥有丰富的文化资源（古建筑、非遗技艺、传说故事），但这些资源是**静态、分散**的。都市人群渴望情感疗愈，但**缺乏一个能将文化体验转化为个性化心理干预的媒介**。

### AI 桥梁作用
项目将 AI 定位为"**文化翻译官**"与"**情感共鸣师**"：
- ✅ 读懂用户的心（情感测评）
- ✅ 读懂村落的文化（文化知识库）
- ✅ 将二者精准匹配（智能叙事）
- ✅ 创造独一无二的疗愈体验（个性化路线）
