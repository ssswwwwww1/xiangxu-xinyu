# 数字人疗愈引导系统说明

## 🎯 核心功能升级

数字人"福宝"现在是一个**主动的疗愈引导师**，会根据游客所在位置自动推送个性化的心理疗愈任务和指导。

---

## ✨ 主要特性

### 1. **LBS 位置感知**
- 当游客到达各个景点时，数字人自动检测位置
- 1 秒后主动弹出欢迎消息和任务引导
- 显示当前引导的地点名称

### 2. **剧本化任务引导**
根据您提供的三个剧本，为每个地点配置了完整的疗愈流程：

#### 📍 状元井 - 《外化·命名"风"的故事》
- **故事引导**: 白龙传说，隐喻内心焦虑
- **核心任务**: 为内心的"风"命名（如"催促之风"）
- **具体行动**: 用炭笔在纸片上写下"风"的名字
- **反思提问**: 最近一次它让你"失手"是什么事？
- **疗愈指导**: 静心呼吸，体验井水的"沉静"感

#### 📍 一颗印 - 《寻找独特结果·"第一针"的专注》
- **故事引导**: 刺绣师傅倪源均专注当下的故事
- **核心任务**: 学习平针绣，专注于一针一线
- **具体行动**: 在布上绣下代表新身份的符号
- **反思提问**: 这个"专注的时刻"，在你的生活中是否曾以其他形式出现过？
- **疗愈指导**: 理解"根基稳固方能承载"的寓意

#### 📍 瓦猫工坊 - 《重构·成为"定桥人"》
- **故事引导**: 瓦猫制作工序，寻找内心节奏
- **核心任务**: 亲手绘制一只瓦猫
- **具体行动**: 将写有"风"名的纸片缝于瓦猫符号下
- **反思提问**: 你会如何建立自己的"守护结界"？
- **疗愈指导**: 订立"定桥人仪式"契约

#### 📍 舞龙广场 - 《解构·追溯"风"的踪迹》
- **故事引导**: 白龙被感化成为守护神
- **核心任务**: 跟随舞龙队基础训练
- **具体行动**: 在舞动中找到自己的节奏
- **反思提问**: 你故事里的"风"让你采取了什么行动？
- **疗愈指导**: 体验焦虑可以转化为能量

#### 📍 悲悯寺 - 《见证与仪式·安放契约》
- **故事引导**: 悲悯寺见证无数祈愿
- **核心任务**: 分享自己的新故事与身份符号
- **具体行动**: 领取"安神符"，封存新身份符号
- **反思提问**: 你的新角色可以叫什么？
- **疗愈指导**: 订立契约，启动"定桥人仪式"

### 3. **任务弹窗界面**
当游客到达景点时，会显示精美的任务卡片：

**顶部** - 渐变色头部
- 地点图标（🌊状元井、🪡一颗印、🐱瓦猫、🐉舞龙、🏯悲悯寺）
- 任务标题
- 地点名称
- 引导介绍

**内容区域** - 5 个疗愈模块
1. 📖 **故事引导**（紫色）- 讲述文化背景和隐喻
2. ✅ **核心任务**（蓝色）- 主要疗愈任务
3. 🙌 **具体行动**（绿色）- 可操作的行动指南
4. ❓ **反思提问**（琥珀色）- 引导深度思考
5. 💖 **疗愈指导**（粉色）- 心理学指导语

**底部按钮**
- "稍后再看" - 暂时关闭，可随时重新打开
- "我已完成任务" - 确认完成，数字人给予鼓励

### 4. **智能对话系统**
数字人能够理解游客的问题并给出针对性回复：

**关键词识别**：
- "紧张"、"焦虑" → 引导使用"定桥人仪式"
- "美"、"好看" → 共鸣并强化积极体验
- "历史"、"故事" → 分享文化背景
- 其他 → 倾听和陪伴

**快捷问题**（新用户显示）：
- "这里有什么特别的地方吗？"
- "我感觉有点紧张，怎么办？"
- "推荐一个适合拍照的地方"
- "瓦猫有什么寓意？"

### 5. **任务完成反馈**
当游客点击"我已完成任务"时：
```
🎉 太棒了！您已经完成了这个站点的疗愈任务。
这份专注与觉察，将成为您新故事的开始。
继续前行吧，"定桥人"！
```

## 💡 心理学原理应用

### 叙事疗法五步骤

每个地点的任务都遵循叙事疗法的完整流程：

1. **外化对话**（状元井）
   - 将焦虑命名为"风"
   - 分离"人"与"问题"

2. **解构性提问**（舞龙广场）
   - 追溯"风"的踪迹
   - 解构无效行为模式

3. **寻找独特结果**（一颗印）
   - 发现"专注的时刻"
   - 寻找被忽视的自身资源

4. **重构对话**（瓦猫工坊）
   - 建构新身份（"定桥人"）
   - 丰富替代性故事

5. **见证与仪式**（悲悯寺）
   - 社会性见证
   - 仪式化巩固新叙事

## 🔧 技术实现

### 位置检测
```javascript
useEffect(() => {
  if (currentLocation) {
    const taskData = getTaskForLocation(currentLocation);
    setCurrentTask(taskData);
    setShowTask(true);
    
    // 1 秒后主动推送
    setTimeout(() => {
      setMessages(prev => [...prev, {
        type: 'ai',
        content: `✨ 检测到您已到达${currentLocation.name}！${taskData.intro}`,
        emoji: '🌟'
      }]);
    }, 1000);
  }
}, [currentLocation]);
```

### 任务配置
每个地点的任务都配置在 `getTaskForLocation` 函数中，包含：
- title（任务标题）
- intro（引导介绍）
- story（故事引导）
- task（核心任务）
- action（具体行动）
- question（反思提问）
- therapy（疗愈指导）
- icon（地点图标）
