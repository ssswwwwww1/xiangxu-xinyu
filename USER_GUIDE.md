# 乡叙心域 - 使用指南

## 🎯 快速开始

### 最简单的方式（推荐新手）

直接双击打开 **`demo.html`** 文件，即可在浏览器中体验完整功能！

无需安装任何依赖，无需配置环境，开箱即用！

---

## 📱 完整功能体验流程

### 第一步：开始疗愈之旅
1. 打开首页
2. 点击"开始疗愈之旅"按钮
3. 进入情感测评页面

### 第二步：情感测评
1. 回答 5 道情感测评题
2. 每道题选择最符合你当前状态的选项
3. 等待 AI 分析结果（约 2 秒）

**测评维度：**
- 最近一周的整体情绪状态
- 希望获得的疗愈体验
- 偏好的疗愈方式
- 当前最困扰的情绪
- 理想的疗愈节奏

### 第三步：探索个性化路线
AI 会根据你的测评结果生成专属路线：

**三种路线类型：**
1. **宁静心灵之路**（适合焦虑用户）
   - 福安古井 → 瓦猫工坊 → 心灵庭院
   
2. **活力焕发之路**（适合疲惫用户）
   - 刺绣坊 → 古村落广场 → 阳光露台
   
3. **文化探索之路**（适合平衡状态用户）
   - 百年祠堂 → 非遗展示馆 → 古戏台

每个打卡点包含：
- 📍 地点名称
- ⏱️ 预计停留时间
- 📝 活动描述
- 💖 预期情感体验

### 第四步：打卡互动
到达每个打卡点后，进行打卡：

**三种打卡方式：**
1. **拍照打卡** 📷
   - 拍摄当前景点照片
   - 支持重新拍摄
   
2. **语音日记** 🎤
   - 录制你的感受
   - AI 自动转文字
   
3. **文字感悟** ✍️
   - 写下此刻的想法
   - 字数不限

**选择情感标签：**
- 😌 平静
- 😊 喜悦
- 🥺 感动
- 💡 启发
- 🌙 怀念
- 🎯 专注

### 第五步：与数字人互动
随时可以点击"召唤数字人向导"：

**数字人功能：**
- 💬 智能对话
- 🎤 语音交互
- ❓ 快捷问题
- ❤️ 情感支持

**可以问数字人：**
- "这里有什么特别的地方吗？"
- "我感觉有点紧张，怎么办？"
- "推荐一个适合拍照的地方"
- "瓦猫有什么寓意？"

### 第六步：生成心灵游记
完成所有打卡后，AI 会自动生成：

**心路影像诗包含：**
- 📖 专属诗歌（基于你的体验生成）
- 📊 旅程统计（打卡点数量、总时长）
- 💖 情感分布图（各种情绪的占比）
- 📸 打卡记录时间线

**分享和保存：**
- 🔗 分享到朋友圈
- 🖼️ 生成精美图片
- 🎵 配乐朗诵版本
- ⬇️ 下载线装手账 PDF

## 💡 使用技巧

### 获得最佳体验的建议

1. **真实表达情感**
   - 测评时选择最符合真实状态的选项
   - 打卡时记录真实的感受
   - AI 会根据你的真实情感提供更精准的推荐

2. **充分与数字人互动**
   - 遇到疑问随时询问
   - 分享你的感受和想法
   - 数字人会提供情感支持和建议

3. **完整体验所有打卡点**
   - 每个打卡点都有独特的疗愈活动
   - 完成所有打卡才能获得完整的心路影像诗
   - 不要匆忙，慢慢体验每个瞬间

4. **尝试不同的打卡方式**
   - 拍照：记录视觉美景
   - 语音：捕捉当下感受
   - 文字：深度思考和反思
   - 三种方式结合，体验更丰富

5. **保存和分享你的游记**
   - 心路影像诗是独一无二的情感资产
   - 可以下载保存或分享给朋友
   - 未来可以回顾这次疗愈之旅

## 🔧 技术使用指南

### 开发者模式

如果你想修改或扩展项目：

#### 1. 修改页面内容

编辑 `src/pages/` 目录下的文件：
- `HomePage.tsx` - 首页内容
- `AssessmentPage.tsx` - 测评题目和选项
- `JourneyPage.tsx` - 路线和打卡点信息
- `CheckInPage.tsx` - 打卡功能
- `MemoryPage.tsx` - 游记生成
- `DigitalHumanPage.tsx` - 数字人对话逻辑

#### 2. 添加新的村落

在 `JourneyPage.tsx` 中添加新的路线配置：

```javascript
const routes = {
  yourNewVillage: {
    name: '你的村落名称',
    description: '村落描述',
    color: 'from-blue-500 to-cyan-500',
    stops: [
      {
        id: 1,
        name: '打卡点 1',
        type: 'landmark',
        duration: '30 分钟',
        description: '描述',
        activity: '活动',
        emotion: '平静'
      }
      // 更多打卡点...
    ]
  }
}
```

#### 3. 自定义 AI 回复

在 `DigitalHumanPage.tsx` 中修改 `aiResponses` 数组：

```javascript
const aiResponses = [
  {
    keywords: ['你的关键词'],
    responses: ['AI 回复内容 1', 'AI 回复内容 2'],
    emotion: 'happy'
  }
]
```

#### 4. 修改样式

编辑 `tailwind.config.js` 自定义颜色和主题：

```javascript
theme: {
  extend: {
    colors: {
      primary: {
        // 自定义主题色
      }
    }
  }
}
```

## ❓ 常见问题

### Q1: 测评结果不准确怎么办？
A: 测评结果基于你的真实选择。如果状态发生变化，可以重新测评。

### Q2: 可以跳过某些打卡点吗？
A: 建议完整体验所有打卡点，但技术上可以跳过。不过会影响心路影像诗的完整性。

### Q3: 数字人能回答所有问题吗？
A: 数字人目前预设了一些常见问题。如果遇到无法回答的问题，可以尝试换个问法。

### Q4: 生成的诗歌可以修改吗？
A: 当前版本诗歌由 AI 自动生成。你可以在分享前截图保存，然后手动编辑。

### Q5: 可以在手机上使用吗？
A: 可以！项目采用响应式设计，完美适配手机、平板和电脑。

### Q6: 数据会保存多久？
A: 当前版本数据保存在浏览器本地（localStorage）。清除浏览器缓存会丢失数据。

### Q7: 可以多人同时使用吗？
A: 可以。每个用户的浏览器会独立保存数据。
